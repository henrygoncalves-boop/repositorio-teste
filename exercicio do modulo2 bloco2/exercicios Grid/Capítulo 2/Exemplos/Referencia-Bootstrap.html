O que é o Bootstrap

Bootstrap é um framework CSS — uma biblioteca de estilos e componentes já prontos que você adiciona ao seu projeto. Em vez de escrever CSS para cada botão, card ou menu, você usa as classes do Bootstrap diretamente no HTML.

📦

Framework vs CSS puro: CSS puro dá controle total, mas você escreve tudo. Bootstrap acelera o desenvolvimento com componentes prontos, mas você fica limitado ao visual padrão dele — a menos que personalize.

As principais vantagens do Bootstrap:

  • Responsividade automática — o grid se adapta a qualquer tela com classes simples
  • Componentes prontos — botões, alertas, menus, cards, modais e muito mais
  • Consistência visual — tudo segue o mesmo padrão de design
  • Velocidade — você prototipa um site em horas, não dias

Como Instalar

A forma mais simples é via CDN — você adiciona um link no <head> e o Bootstrap já está disponível, sem precisar instalar nada.

html
<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1">

  <!-- Bootstrap CSS -->
  <link
    href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/css/bootstrap.min.css"
    rel="stylesheet">

  <title>Meu Site</title>
</head>
<body>

  <!-- Seu conteúdo aqui -->

  <!-- Bootstrap JS (necessário para componentes interativos) -->
  <script src="https://cdn.jsdelivr.net/npm/bootstrap@5.3.0/dist/js/bootstrap.bundle.min.js"></script>
</body>
</html>
⚠️

A tag <meta name="viewport"> é obrigatória para o Bootstrap funcionar corretamente em dispositivos móveis. Não esqueça dela.

Para usar o grid, você precisa de três elementos:

  • .container — centraliza e limita a largura do conteúdo
  • .row — cria uma linha na grade
  • .col-* — define quantas colunas o elemento ocupa (1 a 12)
exemplos de divisão em 12 colunas

<div class="row">

<div class="row">

  <div class="col-4">Um terço</div>
</div>

<!-- Sidebar + conteúdo: 3 + 9 = 12 -->
<div class="row">
  <div class="col-3">Sidebar</div>

💡

A soma das colunas dentro de uma .row deve sempre dar 12. Se der menos, sobra espaço. Se der mais de 12, o que passar quebra para a linha de baixo.

col sem número — divisão automática

Se você usar só col sem número, o Bootstrap divide o espaço igualmente entre todos os filhos automaticamente.

col sem número — divisão automática
<div class="row">
  <div class="col">Auto</div>
  <div class="col">Auto</div>
  <div class="col">Auto</div>
  <div class="col">Auto</div>
</div>

/* Cada col ocupa 3 colunas

Breakpoints e Classes Responsivas

A grande mágica do Bootstrap é poder definir layouts diferentes para cada tamanho de tela usando apenas classes no HTML — sem escrever media queries.

Combinando breakpoints

Você combina as classes para definir comportamentos diferentes por tela. O Bootstrap aplica a classe do breakpoint mais próximo abaixo.

layout responsivo com breakpoints
<div class="row">
  <div class="col-12 col-md-6 col-lg-4">
    Card
  </div>
</div>

/* Mobile:  col-12 → 1 coluna
   Tablet:  col-md-6 → 2 colunas
   Desktop: col-lg-4 → 3 colunas */

Classes Utilitárias

Além do grid e dos componentes, o Bootstrap tem classes utilitárias — classes que fazem uma coisa só, mas que você usa em qualquer elemento. Elas substituem CSS simples que você escreveria do zero.

Espaçamento — m e p

Controla margin e padding sem escrever CSS. A lógica é: {propriedade}{lado}-{tamanho}.

ClasseO que faz
m-3margin em todos os lados (tamanho 3)
mt-2margin-top (t = top)
mb-4margin-bottom (b = bottom)
mx-automargin-left e margin-right: auto — centraliza
p-3padding em todos os lados
px-4padding-left e padding-right (x = horizontal)
py-2padding-top e padding-bottom (y = vertical)

Texto

ClasseO que faz
text-centertext-align: center
text-endtext-align: right
fw-boldfont-weight: bold
text-primaryCor azul do Bootstrap
text-mutedTexto cinza apagado
fs-4Tamanho de fonte (1 = maior, 6 = menor)

Display e Flexbox

ClasseO que faz
d-flexdisplay: flex
d-nonedisplay: none
d-md-blockdisplay: block apenas em tablets e acima
justify-content-centerjustify-content: center
align-items-centeralign-items: center
gap-3gap entre os itens flex
💡

As classes de display também aceitam breakpoints: d-none d-md-block esconde o elemento no mobile e mostra a partir do tablet. Isso é muito usado para mostrar/esconder menus diferentes por dispositivo.

Bootstrap é uma ferramenta de produtividade, não um substituto para aprender CSS. Entender Flexbox, Grid e Box Model é fundamental — o Bootstrap é apenas um atalho que usa esses conceitos por baixo dos panos.
