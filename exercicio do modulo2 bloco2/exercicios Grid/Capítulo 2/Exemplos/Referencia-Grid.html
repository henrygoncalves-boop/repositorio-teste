CSS Grid

O Grid é o sistema de layout bidimensional do CSS — você define colunas e linhas ao mesmo tempo, e posiciona os elementos na grade como quiser.

O que é o Grid e como funciona

O CSS Grid cria uma grade invisível no contêiner pai — você define quantas colunas e linhas ela tem, e os filhos se encaixam automaticamente nas células ou podem ser posicionados em qualquer lugar da grade.

📐

Flexbox vs Grid: Flexbox organiza elementos em uma direção (linha ou coluna). Grid organiza em duas direções ao mesmo tempo (linhas e colunas). Use Flexbox para componentes, Grid para o layout da página.

Para ativar o Grid, basta aplicar display: grid no elemento pai:

display: grid — sem definir colunas
.container {
  display: grid;
}

/* Sem grid-template-columns,
   cada filho ocupa uma linha —
   comportamento padrão */

Definindo Colunas e Linhas

grid-template-columns

Define quantas colunas a grade tem e qual o tamanho de cada uma. Cada valor separado por espaço representa uma coluna.

3 colunas iguais
.container {
  display: grid;
  grid-template-columns: 1fr 1fr 1fr;
  gap: 10px;
}

grid-template-rows

Funciona igual ao grid-template-columns, mas para as linhas. Na maioria dos casos você não precisa definir — as linhas crescem automaticamente para acomodar o conteúdo.

gap

Define o espaço entre as células da grade — tanto entre colunas quanto entre linhas. Você pode usar um valor só (gap: 16px) ou dois separados (gap: linha coluna).

gap — espaço entre as células
/* Mesmo gap em todos os lados */
gap: 16px;

/* Gap diferente por eixo */
gap: 24px 8px;

repeat()

Atalho para repetir o mesmo valor várias vezes, evitando repetição no código.

repeat() — atalho para colunas repetidas
/* Sem repeat — repetitivo */
grid-template-columns: 1fr 1fr 1fr 1fr;

/* Com repeat — mais limpo */

A Unidade fr

A unidade fr (fraction — fração) representa uma parte do espaço disponível no contêiner. É o coração do Grid responsivo.

O browser calcula o espaço total, desconta o gap e divide proporcionalmente entre as colunas que usam fr.

fr — divisão proporcional do espaço
/* 3 partes iguais */
grid-template-columns: 1fr 1fr 1fr;

/* Proporções diferentes */
grid-template-columns: 1fr 2fr 1fr;
/* Total: 4fr
   Col 1: 25%, Col 2: 50%, Col 3: 25% */

/* Misturando px e fr */
grid-template-columns: 200px 1fr;
/* Col 1: 200px fixos
   Col 2: todo o espaço que sobrar */

Você pode misturar fr com px livremente. O browser reserva o espaço fixo primeiro e divide o que sobrar com fr. Por exemplo: 200px 1fr — 200px para o sidebar, o resto para o conteúdo.

Posicionando Itens na Grade

Por padrão, os itens se encaixam automaticamente nas células da grade. Mas você pode fazer um item ocupar mais de uma coluna ou linha.

grid-column e grid-row

Definem em quais linhas da grade o item começa e termina. A grade tem linhas de grade numeradas — uma grade de 3 colunas tem 4 linhas de grade (1, 2, 3, 4).

grid-column — item ocupando múltiplas colunas
.destaque {
  grid-column: 1 / 3;
  /* da linha 1 até a linha 3
     = ocupa 2 colunas */
}

/* Atalho com span */
.destaque {
  grid-column: span 2;
  /* ocupa 2 colunas a partir
     de onde está */
}

/* -1 = última linha da grade */
.full {
  grid-column: 1 / -1;
  /* ocupa todas as colunas */
}

Grid Areas — Nomeando Regiões

O grid-template-areas permite nomear regiões da grade com palavras, tornando o layout muito mais fácil de ler e entender.

grid-template-areas — layout completo de página
.pagina {
  display: grid;
  grid-template-columns: 1fr 2fr;
  grid-template-areas:
    "header  header"
    "sidebar main"
    "footer  footer";
  gap: 8px;
}

header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
main    { grid-area: main; }
footer  { grid-area: footer; }

💡

O grid-template-areas funciona como um mapa visual do layout. Cada string entre aspas representa uma linha da grade, e cada palavra representa uma célula. Nomes repetidos fazem o item ocupar várias células.

Tornando o layout responsivo com Media Query

Uma das grandes vantagens do Grid Areas é como fica simples reorganizar o layout para mobile:

css
/* Desktop — sidebar + conteúdo lado a lado */
.pagina {
  display: grid;
  grid-template-columns: 1fr 2fr;
  grid-template-areas:
    "header  header"
    "sidebar main"
    "footer  footer";
  gap: 16px;
}

/* Mobile — tudo em coluna única */
@media (max-width: 768px) {
  .pagina {
    grid-template-columns: 1fr;
    grid-template-areas:
      "header"
      "main"
      "sidebar"
      "footer";
  }
}

Resumo das Propriedades

Propriedades do Contêiner (pai)

PropriedadeO que faz
display: gridAtiva o Grid no contêiner
grid-template-columnsDefine o número e tamanho das colunas
grid-template-rowsDefine o número e tamanho das linhas
grid-template-areasNomeia regiões da grade com palavras
gapEspaço entre as células
justify-itemsAlinha os itens horizontalmente dentro das células
align-itemsAlinha os itens verticalmente dentro das células

Propriedades do Item (filho)

PropriedadeO que faz
grid-columnDefine quais colunas o item ocupa (1 / 3 ou span 2)
grid-rowDefine quais linhas o item ocupa
grid-areaAssocia o item a uma área nomeada no contêiner
justify-selfAlinhamento horizontal individual
align-selfAlinhamento vertical individual
